# Лабораторная работа 6: 3D модель буквы "И"

## Описание проекта

Интерактивное веб-приложение для визуализации и преобразования 3D модели буквы "И" с использованием HTML5 Canvas и JavaScript. Проект демонстрирует основные операции 3D-графики: масштабирование, перенос, вращение и ортографические проекции.

## Функциональные возможности

### 1. Основные возможности
- **3D визуализация**: Отображение трехмерной модели буквы "И" с системой координат
- **Интерактивное управление**: Ползунки для управления преобразованиями в реальном времени
- **Ортографические проекции**: Три стандартные проекции (OXY, OXZ, OYZ)
- **Матричные преобразования**: Отображение текущей матрицы преобразования

### 2. Преобразования модели
- **Масштабирование**: По осям X, Y, Z (от 0.1 до 3)
- **Перенос**: По осям X, Y, Z (от -200 до 200)
- **Вращение**: Вокруг осей X, Y, Z (0-360 градусов)

### 3. Управление
- **Сброс параметров**: Возврат к исходному состоянию
- **Случайное положение**: Генерация случайных значений преобразований
- **Реальное отображение**: Все изменения отображаются мгновенно

## Структура проекта

### HTML-структура
- Основной контейнер с сеткой для расположения элементов
- Canvas-элементы для 3D вида и проекций
- Панели управления с ползунками и кнопками
- Отображение матрицы преобразования

### CSS-стили
- Адаптивная сетка (Grid Layout)
- Стилизованные панели управления
- Цветовое кодирование групп элементов
- Четкая визуальная иерархия

### JavaScript-компоненты

#### 1. Класс `Vector3`
- Хранение трехмерных координат
- Методы для трансформации и клонирования

#### 2. Класс `Letter3D`
- Определение вершин и ребер буквы "И"
- Хранение исходных и преобразованных данных
- Сетка модели с 20 вершинами и 30 ребрами

#### 3. Класс `TransformationMatrix`
- Реализация 4x4 матрицы преобразований
- Методы для масштабирования, переноса и вращения
- Матричное умножение
- Форматированный вывод матрицы

#### 4. Класс `Renderer`
- Отрисовка на Canvas
- Проекции 3D в 2D
- Отображение осей координат
- Рендеринг ребер модели

#### 5. Класс `App`
- Главный класс приложения
- Управление контроллерами
- Координация всех компонентов
- Обработка пользовательского ввода

## Математические основы

### Матрица преобразования
Применяется однородная матрица 4×4:
```
[ sx*cosRY*cosRZ - sx*sinRZ*sinRY*cosRX + sx*sinRX*sinRY  tx ]
[ sy*cosRX*sinRZ + sy*sinRX*sinRY*cosRZ                     ty ]
[ sz*sinRX*cosRZ - sz*sinRZ*cosRX                           tz ]
[ 0                        0                        0        1 ]
```

### Проекции
1. **3D вид**: Аксонометрическая проекция с коэффициентом 0.7
2. **OXY**: Ортографическая проекция на плоскость XY (вид сверху)
3. **OXZ**: Ортографическая проекция на плоскость XZ (вид спереди)
4. **OYZ**: Ортографическая проекция на плоскость YZ (вид сбоку)

## Инструкция по использованию

### 1. Управление преобразованиями
- **Масштабирование**: Используйте ползунки Scale X/Y/Z
- **Перенос**: Используйте ползунки Translate X/Y/Z
- **Вращение**: Используйте ползунки Rotate X/Y/Z

### 2. Кнопки управления
- **Сброс**: Возвращает модель в исходное состояние
- **Случайное положение**: Генерирует случайные значения преобразований

### 3. Визуализация
- Основное окно показывает 3D вид с осями координат
- Три дополнительных окна показывают ортографические проекции
- Матрица преобразования отображается в реальном времени

## Технические особенности

### Алгоритмы
1. **Порядок преобразований**: Перенос → Вращение → Масштабирование
2. **Проекция 3D→2D**: x' = cx + (x - z*0.7)*scale, y' = cy - (y - z*0.7)*scale
3. **Ортографические проекции**: Отбрасывание одной координаты

### Оптимизации
- Клонирование вершин для сохранения оригинала
- Единоразовое вычисление матрицы преобразования
- Пакетная отрисовка ребер
