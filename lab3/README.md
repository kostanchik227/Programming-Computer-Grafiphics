# Лабораторная работа 3: Обработка цифровых изображений

## Описание проекта

Веб-приложение для обработки изображений с реализацией методов увеличения резкости, линейного контрастирования и эквализации гистограммы. Приложение разработано с использованием HTML5 Canvas и JavaScript.

## Функциональность

### Основные возможности:
- **Загрузка изображений** с локального устройства
- **Увеличение резкости** с регулируемой интенсивностью (0-200%)
- **Линейное контрастирование** с автоматическим определением диапазона яркости
- **Эквализация гистограммы** в двух вариантах: RGB и HSV
- **Визуализация гистограмм** исходного и обработанного изображений
- **Графики преобразований** для методов контрастирования

### Алгоритмы обработки:

#### 1. Увеличение резкости
- Используется высокочастотный фильтр (лапласиан)
- Ядро свертки
- Регулировка интенсивности через слайдер

#### 2. Линейное контрастирование
Формула преобразования: 
```
I' = ((I - I_min) × 255) / (I_max - I_min)
```
где I_min и I_max - минимальное и максимальное значения яркости в изображении.

#### 3. Эквализация гистограммы
- **RGB метод**: отдельная обработка каждого цветового канала
- **HSV метод**: преобразование в HSV, эквализация канала Value (яркости), обратное преобразование в RGB

## Использование

1. **Загрузка изображения**
   - Нажмите "Выберите файл" и выберите изображение
   - Нажмите "Загрузить изображение"

2. **Применение фильтров**
   - **Увеличение резкости**: установите интенсивность слайдером и нажмите кнопку
   - **Линейное контрастирование**: нажмите соответствующую кнопку
   - **Эквализация гистограммы**: выберите RGB или HSV метод

3. **Просмотр результатов**
   - Исходное и обработанное изображения отображаются рядом
   - Гистограммы обновляются автоматически
   - Графики показывают примененные преобразования

4. **Сброс**
   - Используйте кнопку "Сбросить" для возврата к исходному изображению

## Технические детали

### Поддерживаемые форматы:
- JPEG, PNG, GIF, BMP и другие форматы изображений

### Ограничения:
- Изображения автоматически масштабируются до ширины 600px для удобства просмотра
- Обработка выполняется в браузере, производительность зависит от размера изображения

### Интерфейс:
- Интуитивно понятное управление
- Визуальная обратная связь через гистограммы и графики
- Адаптивный дизайн для различных размеров экрана
