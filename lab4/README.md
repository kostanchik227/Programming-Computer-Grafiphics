# README: Лабораторная работа №4 - Базовые растровые алгоритмы

## Описание проекта
Интерактивное веб-приложение для визуализации и изучения базовых растровых алгоритмов, используемых в компьютерной графике для растеризации отрезков и кривых.

## Основные функции
1. **6 алгоритмов растеризации:**
   - Пошаговый алгоритм
   - Алгоритм ЦДА (Цифровой Дифференциальный Анализатор)
   - Алгоритм Брезенхема (линия)
   - Алгоритм Брезенхема (окружность)
   - Алгоритм Кастла-Питвея (кривые Безье)
   - Алгоритм Ву (сглаживание)

2. **Интерактивные возможности:**
   - Интерактивный холст с координатной сеткой
   - Настройка параметров отрисовки
   - Визуализация шагов выполнения алгоритмов
   - Отображение временных характеристик
   - Горячие клавиши для быстрого доступа

## Технологии
- HTML5 Canvas для графики
- Чистый JavaScript (без внешних библиотек)
- CSS3 для стилизации и анимаций
- Адаптивный дизайн

## Инструкция по использованию

### Управление интерфейсом
1. **Выбор алгоритма:** Нажмите на одну из 6 кнопок в верхней панели
2. **Настройка параметров:**
   - Введите координаты начальной и конечной точек
   - Для окружности задайте радиус
   - Для кривых Безье выберите количество контрольных точек
3. **Настройка отображения:**
   - Масштаб: регулирует размер пикселей (5-20px)
   - Размер сетки: настраивает частоту линий сетки (5-50 единиц)
4. **Основные действия:**
   - "Построить" - выполнить выбранный алгоритм
   - "Очистить" - очистить холст

### Особые режимы
- **Для алгоритма Кастла-Питвея:** кликайте на холст для добавления контрольных точек кривой
- **Алгоритм Ву:** автоматически применяет сглаживание (антиалиасинг)

### Горячие клавиши
- **1-6:** Быстрый выбор алгоритма
- **Пробел:** Выполнить построение
- **C:** Очистить холст

## Теоретическая информация

### Алгоритмы и их особенности

#### 1. Пошаговый алгоритм
- **Принцип:** Вычисление точек по уравнению прямой y = kx + b
- **Сложность:** O(max(|dx|, |dy|))
- **Особенности:** Простейшая реализация, но требует вычислений с плавающей точкой

#### 2. Алгоритм ЦДА
- **Принцип:** Использование приращений для вычисления следующих точек
- **Сложность:** O(max(|dx|, |dy|))
- **Особенности:** Более эффективен чем пошаговый, но также использует вещественные числа

#### 3. Алгоритм Брезенхема (линия)
- **Принцип:** Использование целочисленных операций и анализа ошибки
- **Сложность:** O(max(|dx|, |dy|))
- **Преимущества:** Только целочисленные операции, высокая скорость

#### 4. Алгоритм Брезенхема (окружность)
- **Принцип:** Использование симметрии окружности и целочисленных операций
- **Сложность:** O(радиус)
- **Особенности:** Рисует только 1/8 окружности, остальное достраивается симметрично

#### 5. Алгоритм Кастла-Питвея
- **Принцип:** Рекурсивное разделение контрольного многоугольника для построения кривых Безье
- **Сложность:** O(n × 2^k), где n - количество точек, k - глубина рекурсии
- **Применение:** Сглаженные кривые, компьютерная графика, проектирование

#### 6. Алгоритм Ву
- **Принцип:** Использование антиалиасинга через изменение прозрачности пикселей
- **Сложность:** O(max(|dx|, |dy|))
- **Результат:** Сглаженные линии без "лестничного эффекта"

## Система координат
- **Центр:** (0,0) находится в центре холста
- **Ось X:** направлена вправо
- **Ось Y:** направлена вверх (противоположно экранным координатам)
- **Диапазон:** от -250 до 250 по каждой оси
- **Сетка:** привязана к целочисленным координатам

## Требования к браузеру

  - HTML5 Canvas
  - ES6 JavaScript
  - CSS Grid и Flexbox

## Особенности реализации
1. **Модульность:** Все алгоритмы реализованы как отдельные функции
2. **Производительность:** Минимальное использование перерисовок
3. **Интерактивность:** Реагирование на изменения в реальном времени
4. **Образовательный фокус:** Подробный вывод шагов выполнения
