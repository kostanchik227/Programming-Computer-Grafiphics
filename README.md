# Лабораторная работа №1: Цветовые модели RGB, XYZ, Lab

**Вариант:** RGB ↔ XYZ ↔ Lab

---

## 1. Цель работы

Изучить цветовые модели RGB, XYZ и CIE Lab, реализовать приложение для интерактивного преобразования цвета между ними, исследовать преобразования и особенности цветовых пространств.

---

## 2. Постановка задачи

Необходимо разработать веб-приложение, позволяющее пользователю:

- выбирать цвет из палитры;
- задавать точные значения цветовых координат вручную;
- изменять цвет с помощью ползунков;
- отображать цвет в трёх моделях одновременно: RGB, XYZ, Lab;
- автоматически пересчитывать значения во всех моделях при изменении любой координаты;
- выводить предупреждения при выходе цвета за доступный диапазон sRGB.

Приложение должно быть доступно онлайн, а исходный код размещён в репозитории GitHub.

---

## 3. Краткое описание цветовых моделей

### 3.1 RGB
- Аддитивная модель: смешение компонентов Red, Green, Blue
- Используется в мониторах, графике и веб-разработке
- Диапазоны: `R, G, B ∈ [0; 255]`

### 3.2 XYZ (CIE 1931)
- Линейная модель, основанная на измерениях чувствительности человеческого глаза
- Используется как промежуточная модель для переводов
- Белая точка: D65
- Диапазон значений X, Y, Z зависит от конкретного цвета (обычно до ~95–110)

### 3.3 Lab (CIELAB)
- Перцептуальная модель, приближенная к восприятию человека
- Диапазоны:  
  `L ∈ [0; 100]`  
  `a ∈ [-128; 127]`  
  `b ∈ [-128; 127]`

## 4. Используемые технологии

- **HTML + CSS** — адаптивная верстка  
- **JavaScript** — полные математические преобразования между моделями  
- Поддержка линейного преобразования: `sRGB → XYZ → Lab` (и обратно)  
- Поддержка HEX ввода цвета  

---

## 5. Функционал приложения

- Ползунки для всех каналов RGB/XYZ/Lab  
- Поля ручного ввода значений  
- Цветовой HEX-пикер  
- Динамическая панель выбранного цвета  
- Кнопка копирования HEX  
- Система предупреждений при выходе цвета за гамму sRGB  
- Автоматический пересчёт при изменении любого параметра  

---

## 6. Структура интерфейса

- **Левая панель:** выбор цвета, HEX поле, быстрый выбор модели  
- **Правая панель:** три блока моделей (RGB/XYZ/Lab), каждый содержит:
  - ползунок компоненты  
  - числовое поле  
  - Swatch-образец с текущим цветом  
  - предупреждение при некорректном значении  

---

## 7. Алгоритмы преобразования

### RGB → XYZ
- Используется стандарт sRGB D65 с гамма-коррекцией

### XYZ → Lab и обратно
- Стандарт CIE 1976 с белой точкой D65

### XYZ → RGB
- Обратное линейное преобразование + обратная гамма-коррекция

> В приложении используются корректные матрицы и функции нелинейности

---

## 8. Проверка корректности работы

- Любое изменение параметра корректно пересчитывает все остальные модели  
- RGB → XYZ → RGB возвращает стабильный цвет (с округлением)  
- Некорректные значения XYZ корректно обрезаются и выдаётся предупреждение  
- HEX всегда синхронизирован с RGB
